# SS-INFRA-GO
This repository provides a unified and streamlined approach to creating Go projects, encompassing features such as Redis, MySQL, tracing, and metrics.

# Features
- [x] gin router middleware
- [x] metrics
- [x] tracing
- [x] log/access log
- [x] response
- [x] config
- [x] mysql
- [x] redis
